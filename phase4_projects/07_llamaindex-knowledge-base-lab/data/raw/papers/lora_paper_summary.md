# LoRA: Low-Rank Adaptation of Large Language Models

## 核心思想
LoRA 通过在预训练权重旁添加低秩分解矩阵来实现参数高效微调（PEFT）。

## 数学表达
h = W_0 * x + B * A * x

其中：
- W_0 ∈ R^{d×k}：预训练权重（冻结）
- B ∈ R^{d×r}，A ∈ R^{r×k}：低秩分解矩阵（可训练）
- r << min(d,k)：秩，通常 r ∈ {4, 8, 16, 32}

## 关键技术细节

### 初始化策略
- A：使用高斯分布初始化
- B：使用零初始化，确保训练开始时 ΔW = BA = 0

### 缩放因子
通常在前向传播时使用 α/r 缩放：
h = W_0 * x + (α/r) * B * A * x

### 应用到哪些权重
论文实验表明，仅对 Attention 的 Q、V 矩阵应用 LoRA 即可达到很好效果。
实际应用中常扩展到 Q、K、V、O 全部四个矩阵。

## 优势
- 参数量减少 10000 倍（例如 175B GPT-3 只需 4.7M 可训练参数）
- 无推理延迟增加（可以合并回原权重：W = W_0 + BA）
- 可插拔式任务切换（不同任务只需切换 A、B 矩阵）
