# DPO: Direct Preference Optimization

## 动机
传统 RLHF 需要三个步骤：
1. 监督微调 (SFT)
2. 训练奖励模型 (RM)
3. 使用 PPO 强化学习优化策略

DPO 将这个过程简化为单步优化，直接使用偏好数据对。

## 核心公式

DPO 损失函数：
L_DPO(π_θ; π_ref) = -E[(x,y_w,y_l)∼D][log σ(β * log(π_θ(y_w|x)/π_ref(y_w|x)) - β * log(π_θ(y_l|x)/π_ref(y_l|x)))]

其中：
- π_θ：待优化的策略模型
- π_ref：参考模型（通常是 SFT 后的模型）
- y_w：偏好回答 (win)
- y_l：非偏好回答 (lose)
- β：温度参数，控制偏离参考模型的程度
- σ：sigmoid 函数

## 优势
- 无需训练单独的奖励模型
- 训练更稳定，无需 PPO 的在线采样
- 计算效率更高
- 理论基础扎实：DPO 的最优解与 RLHF 相同

## 局限性
- 对偏好数据的质量要求更高
- 无法利用未标注的生成样本（PPO 可以）
- β 参数对结果影响敏感
