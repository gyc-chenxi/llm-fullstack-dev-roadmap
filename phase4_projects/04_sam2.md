## 3️⃣ 三种 Prompt 模式

### Point Prompt（单击分割）

```python
# 指定要分割的位置 + 标签（1=前景，0=背景）
input_point = np.array([[400, 300]])  # x, y 坐标
input_label = np.array([1])           # 1=前景

masks, scores, logits = predictor.predict(
    point_coords=input_point,
    point_labels=input_label,
    multimask_output=True,  # 输出 3 个 mask（整体/局部/子物体）
)

# 选得分最高的 mask
best_idx = np.argmax(scores)
mask = masks[best_idx]  # [H, W] 布尔数组

plt.figure(figsize=(12, 6))
plt.subplot(1, 2, 1); plt.imshow(image_rgb); plt.scatter(400, 300, c='r', s=50)
plt.subplot(1, 2, 2); plt.imshow(image_rgb); plt.imshow(mask, alpha=0.5, cmap='jet')
plt.show()
```

### Box Prompt（框选分割）

```python
# box = [x1, y1, x2, y2]

    point_labels=None,
    box=input_box[None, :],
```

## 4️⃣ Mask 后处理

```python
import cv2

def postprocess_mask(mask: np.ndarray, min_area: int = 500) -> np.ndarray:
    """Mask 后处理：去噪 → 填孔 → 平滑 → 过滤小面积"""
    mask_uint8 = (mask * 255).astype(np.uint8)
    
    # 1. 形态学操作：去除噪点
    kernel = cv2.getStructuringElement(cv2.MORPH_ELLIPSE, (5, 5))
    cleaned = cv2.morphologyEx(mask_uint8, cv2.MORPH_OPEN, kernel)   # 去噪
    cleaned = cv2.morphologyEx(cleaned, cv2.MORPH_CLOSE, kernel)     # 填孔
    
    # 2. 连通域分析：过滤小面积
    num_labels, labels, stats, _ = cv2.connectedComponentsWithStats(cleaned)
    result = np.zeros_like(cleaned)
    for i in range(1, num_labels):
        if stats[i, cv2.CC_STAT_AREA] >= min_area:
            result[labels == i] = 255
    
    return result > 0

def mask_quality_report(mask: np.ndarray) -> dict:
    """Mask 质量检测"""
    contours, _ = cv2.findContours(
        (mask * 255).astype(np.uint8), cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE
    )
    return {
        "area": int(mask.sum()),
        "num_contours": len(contours),
        "largest_contour_area": max(cv2.contourArea(c) for c in contours) if contours else 0,
        "edge_smoothness": len(contours[0]) if contours else 0,  # 点数越少越平滑
    }
```

## 5️⃣ SAM 2 + Diffusers Inpaint 联动

```python
from diffusers import StableDiffusionInpaintPipeline

# 1. SAM 2 生成 mask
mask = postprocess_mask(raw_mask)

# 2. 用 mask 做 Inpaint
pipe_inpaint = StableDiffusionInpaintPipeline.from_pretrained(
    "runwayml/stable-diffusion-inpainting",
    torch_dtype=torch.float16 if device == "cuda" else torch.float32,
).to(device)

pil_image = Image.fromarray(image_rgb)
pil_mask = Image.fromarray((mask * 255).astype(np.uint8))

result = pipe_inpaint(
    prompt="a beautiful garden with flowers",
    image=pil_image,
    mask_image=pil_mask,
    num_inference_steps=25,
    guidance_scale=7.5,
)
result.images[0].save("sam2_inpaint_result.png")
```

## 🚨 翻车现场

| 现象 | 原因 | 解决 |
|:-----|:-----|:-----|
| SAM 2 分割不准确 | 图片分辨率不合适 | 确保图片最长边 ≤ 1024px |
| 全图分割太多小物体 | 默认 IoU 阈值太低 | 调高 `pred_iou_thresh=0.88` |
| Point Prompt 没反应 | 坐标是 (x,y) 但用了 (row,col) | 确认 input_point 格式是 [[x, y]] |
| 显存不够 | SAM 2 large 模型太大 | 用 tiny 版（~150MB vs ~2.4GB） |
